# Yet Another Youtube

# Use it locally for development purposes

`$ git clone https://github.com/benjaminaudet/yet-another-youtube.git`

`$ cd yet-another-youtube`

`$ npm install`

`$ npm run start`

# Use for production

`$ git clone https://github.com/benjaminaudet/yet-another-youtube.git`

`$ cd yet-another-youtube`

`$ npm install`

`$ npm run build`
